Maneiras de utilizar e/ou adicionar css

Temos maneiras de utilizar o css ao html.

  1. Pelo inline: quando os estilos são adicionados por um atributo;
  2. Pelo internal: quando o CSS é adicionado na tag head;
  3. Pelo external: quado o css é adicionado através de um arquivo externo, e depois importado no HTMl;

    4. Qual vamos escolher?

    Vamos optar sempre pelo external CSS, quando houver a opção.

    A anatomia do CSS

    Com CSS aplicando CSS a um elemento. Primeiramente devemos selecionar o elemento, isso pode ser feito através da tag do elemento. Depois precisamos colocar as propriedades e os valores, se quisermos mudar a cor de algo utilizamos: color: red, nome da propriedade, dois pontos, valor, ponto e vírgula.

    Inline CSS

    O inline CSS é quando você aplica estilos diretamente em um elemento HTML usando o atributo style, mas pode ser utilizado sem selecionar o elemento, pois é uma tributo diretamente inserido no mesmo.

    Ele nos permite:

  4. Aplicar estilos específicos a um único elemento;
  5. Sobrepor estilos definidos em folhas de estilo externas ou internas;
  6. Ser útil para testes rápidos ou ajustes pontuais;

    7. Porém, o inline CSS não é recomendado para uso extensivo, pois pode tornar o código HTML desorganizado e difícil de manter. É melhor usar folhas de estilo externas ou internas para aplicar estilos de forma consistente em todo o site.

    Internal CSS

    O CSS interno é uma técnica melhor que o inline, vamos colocar todos os estilos na tag head;

  7. As regras precisam também estar entre a tag style;
  8. E através desta maneira, é necessário selecionar o elmento alvo: p{ color: red;}

    External CSS

    Para adicionar CSS com esta técnica precisamos criar um arquivo . css;

  9. Geralmente eles ficam numa pasta chamada css;
  10. E nós o importamos através da tag link;
  11. O link deve estar dentro da tag head;
  12. O link deve ter o atributo rel com o valor stylesheet e as regras que estão no arquivo são aplicadas no HTML.
